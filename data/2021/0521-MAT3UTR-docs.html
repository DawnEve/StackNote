MAT3UTR: Model-based analysis of the trans effect of 3'UTR-shortening¶

Introduction¶

MAT3UTR quantifies the trans effect of 3ʹ-UTR shortening to their affected ceRNA partner. MAT3UTR predicts ceRNA expression change using its 3ʹUS gene(s) expression and 3ʹ-UTR shortening level, miRNA-binding site(s), and miRNA expression(s) based on 3'UTR shortening represses tumor-suppressor genes in trans by disrupting ceRNA crosstalk.)

Please leave your questions and/or feedbacks at Google group for MAT3UTR

Installation¶

Prerequisite for Python codes: python2.7; numpy; scipy; R; Recommend EPD to install those python packages.

Prerequisite for R code: glmnet.

Install MAT3UTR:

tar -xvzf MAT3UTR_VERSION.tar.gz
cd MAT3UTR-VERSION

Instruction

Note that columns in files are separated by tab character.

Step 1: Collect 3'UTR information of genes

threeUTRInfoFromRefSeqInfo.py extracts
1. 3'UTR location information of transcripts (start and end coordinate) with
2. the corresponding gene name based on RefSeq gene models.
Its output can be used as an input file for Step 3 and Step 4 (for --threeUTRInfo).

Options:

-h help
 Show this help message and exit.
-i RefSeqModel, required
 The gene model is RefSeq Genes track downloaded from UCSC.
-o threeUTRInfo, required
 The name of 3'UTR information file that will contain transcript ID, gene name, and the start and the end of 3'UTR.
Note that the start of 3'UTR is always the smaller coordinate value and the end of 3'UTR is the bigger value, regardless of the strand.

Paramters in detail:

    
  
  1. RefSeqModel
    2. This file contains gene model information including their transcript and CDS coordinates, 
    downloaded from the UCSC Genome Browser as in the following.

Example command:

python threeUTRInfoFromRefSeqInfo.py  -i refGene_hg19.txt -o threeUTRInfo.txt

Example output:

This file contains the information of 3'UTR regions of transcripts. 
You may generate this file without referring to RefSeq gene models, since it is generic information as shown below.

Step 2: Collect miRNA binding sites on transcripts

combineMiRSites.py is an example that shows how to collect miRNA binding information for each transcript.
Its output can be used as an input file for Step 3 and Step 4 (for --threeUTRInfo).
Since the collection process highly depends on the format of input data file, we hardcoded some input files we used just for illustration purpose.However, we strongly suggest to write your own code with respect to your input file format.

Options:all input options are hardcoded in the source code. We recommend to write your own code that works for your input file.

-h help
 Show this help message and exit.
-o miRSitesbyTx, required
 This name of the file containing transcript ID, gene name, miRNA ID, and their binding location.

Example command:

python combineMiRSites.py -o miRSitesbyTx.txt

Example output:

This file contains miRNA binding information. 
Each row has transcript ID, gene name, miRNA family name, chromosome, 
start and end coordinate of the binding, and strand information in each column.

Step 3: Build ceRNA network and identify 3'US ceRNA partners¶

buildCeRNETforMAT3UTR.py
1. calculates co-expression in normal samples for transcript pairs with >5 miRNA binding site share
2. identify the ceRNA partners of the given 3'UTR shortening genes
Its output can be used as an input file for Step 4 (for --APA_ceRNA).

Options:

-h help
 Show this help message and exit.
-a threeUTRInfo, required
 This file contains the information of 3'UTR regions of transcripts.
Note that threeUTRInfoFromRefSeqInfo.py can produce this file.
-b geneExprNormal, required
 This file contains expression level of genes in normal condition (in log2).
-c miRSitesbyTx, required
 This file contains transcript ID, gene name, miRNA ID, and their binding location.
-d miRToInclude, optional
 This file contains miRNA IDs to be considered in this analysis, one in a row.
-e threeUS, required
 This file contains information of significant 3'UTR shortening genes, namely transcript ID, alternative polyadenylation site location, PDUI value in tumor and normal, respecctively.
-o APA_ceRNA, required
 This file contains a gene and significant 3'UTR shortening genes the gene is connected to in normal condition.
Note that this file can be used for MAT3UTR.py (for --APA_ceRNA).

Paramters in detail:

    
  
  1. threeUTRInfo 
    2. This file contains the information of 3'UTR regions of transcripts. 
    Step 1 can generate this file.
  
  
  2. geneExprNormal 
    3. This file contains expression level of genes in log2, one row for each gene and one column for each sample.
  
  
  3. miRSitesbyTx 
    4. This file contains miRNA binding information. 
    Each row has transcript ID, gene name, miRNA family name, chromosome, start and end coordinate of the binding
    and strand information. Your implementation in accordance of Step 2 can generate this file. 
  
  
  4. miRToInclude 
    5. Each row has miRNA ID that we will consider in the analysis. 
    We used moderately expressed miRNAs (>1 and <100 in FPM) in this file. 
  
  
  5. threeUS 
    6. Each row contains information of significant 3'UTR shortening gene, namely transcript ID, 
    alternative polyadenylation site location, PDUI value in tumor and normal, respecctively. 
    You can extract this information from DaPars result.

Example command:

python buildCeRNETforMAT3UTR.py -a threeUTRInfo.txt -b TCGAExprNormal.txt -c miRSitesbyTx.txt \
-d miRsNotHighLow.txt -e threeUS_TCGA.txt -o APA_ceRNA.txt

Example output:

Each row represents a ceRNA partner (the first column) and its connected 3'UTR shortening genes (the second column).

Step 4: Estimate 3'UTR shortening trans effect for each gene and miRNA.¶

MAT3UTR.py will use information about genes (gene model, gene expression, and gene differential expression), ceRNAs, and microRNAs (microRNA family, microRNA site, and microRNA expression information) to generate a mtrix file containing APA trans effect on each gene contributed from each miRNA.

Options:

-h help
 Show this help message and exit.
-a threeUTRInfo, required
 This file contains the information of 3'UTR regions of transcripts.
Step 1 can produce this file.
-b geneExprNormal, required
 This file contains expression level of genes in normal condition (in log2).
-c geneExprTumor, required
 This file contains expression level of genes in tumor conditions (in log2).
-d APA_CERNA, required
 gene name as ceRNA partners and their 3'UTR shortening genes and the 3'UTR shortening genes separated by comma.
-e threeUS, required
 gene name whose 3'UTR is shortened, its pUTR position, PDUI in tumor, PDUI in normal.
-f miRSitesbyTx, required
 This file contains transcript ID, gene name, miRNA ID, and their binding location (in the gene).
-g miRNAExpr, required
 miRNA name, (averaged) expression (FPM in log2) in normal and tumor.
-o matFile, required
 MAT3UTR estimation of 3'US trans effect for each gene (row) and each miRNA (column). Note that it will be used for regressMAT3UTR.R.

Paramters in detail:

    
  
  1. threeUTRInfo 
    2. This file contains information of 3'UTR regions of transcripts. You can generate this file from threeUTRInfoFromRefSeqInfo.py.
  
  
  2. geneExprNormal, geneExprTumor
    3. This file contains FPKM values of genes in normal and tumor conditions; 
    Each row represent log2(FPKM) values of each gene starting with its name. 
  
  
  3. APA_CERNA 
    4. This file contains ceRNA partners to 3'UTR shortening genes. 
    Each row represent a ceRNA partners (the first column) and its connected 3'UTR shortening genes (the second column). 
    Step 3 can produce this file.
  
  
  4. threeUS 
    5. Each row contains information of significant 3'UTR shortening gene, namely transcript ID, 
    alternative polyadenylation site location, PDUI value in tumor and normal, respecctively. 
    You can extract this information from DaPars result.
  
  
  5. miRSitesbyTx 
    6. This file contains information of miRNA targeting genes. 
    Each row has transcript ID, gene name, miRNA family name, chromosome, start and end coordiantes, and strand information. 
    Your implementation in accordance of Step 2 can generate this file. 
  
  
  6. miRNAExpr 
    7.  This file contains miRNA expression (miRNA family ID, its expression in normal, expression in tumor).

Example command:

python MAT3UTR.py -a threeUTRInfo.txt -b TCGAExprNormal.txt -c TCGAExprTumor.txt -d APA_ceRNA.txt -e threeUS_TCGA.txt \
-f miRSitesbyTx.txt -g miRsNotHighLow.txt -o MAT3UTR.txt

Example output:

This file contains MAT3UTR's estimation of 3'US trans effect on each gene (row) from each miRNA (column).
Note that the rightmost column should have log2 fold change of the gene expression change (tumor vs. normal). Please note that this is a sparse matrix.

Step 4-2: (Optional) Extract differentially expressed genes from the result of Step 4.¶

extractDEgenes.py simply extracts differentially expressed gene information from MAT3UTR result from previous step.

Options:

-h help
 Show this help message and exit.
-i Differentially expressed genes, required
 Each gene in a row.
For TCGA breast cancer data analysis in the paper, differential expression analyses were carried out by edgeR (version 3.8.6) (tumor samples versus normal samples) with false discovery rate (FDR) control at 0.05.
-m MAT3UTR, required
 MAT3UTR estimation from Step4.
-o MAT3UTRwithDE, required
 MAT3UTR estimation selected for genes in the input file.

Paramters in detail:

    
  
  1. Differentially expressed genes
    2. This file contains names of differentially expressed genes. 
  
  
  2. MAT3UTR
    3. This file is a result from Step 4.

Example command:

python extractDEgenes.py  -i DEgenes_TCGA.txt -m MAT3UTR.txt -o MAT3UTRwithDE.txt

Example output:

This file is a result from Step 4 where only differentially expressed genes are selected.

Step 5: Conduct ridge regression on Step 4 result.¶

Run regressMAT3UTR.R with the result from Step1 by following the procedure below.

  1. Place the result of Step 1 in the same folder as regressMAT3UTR.R
  2. Set MATin variable in the script as the name of the result file (e.g. MAT_TCGA.txt)
  3. Set MATout variable in the script as the name of the file for MAT3UTR score for each gene.
  4. Run regressMAT3UTR.R

Example output:

Each row has gene name, response (log2 fold change (tumor vs. normal)) and the estimated MAT3UTR score.
